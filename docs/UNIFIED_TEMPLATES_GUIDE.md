# 统一模板管理系统使用指南

## 概述

统一模板管理系统将原有的"普通模板"和"高级模板"整合到一个界面中，支持两种模式的智能切换。

**访问地址**：`http://127.0.0.1:5000/templates/unified`

## 两种模式

### 简单模式 🎯

适合新手用户，通过可视化表单配置模板。

**功能特点**：
- ✅ 可视化表单，无需编写代码
- ✅ 固定结构：片头 → 主视频 → 片尾
- ✅ 支持字幕配置
- ✅ 支持文字叠加

**适用场景**：
- 标准化的视频制作流程
- 只需要简单的片头片尾
- 快速创建模板

### 高级模式 🚀

适合高级用户，通过Timeline JSON灵活配置。

**功能特点**：
- ✅ 完全自定义Timeline结构
- ✅ 支持占位符系统（`$main_video`, `$mainSubtitleDepart`等）
- ✅ 灵活的主视频位置配置
- ✅ 支持复杂的视频编排

**适用场景**：
- 需要自定义视频结构
- 需要使用占位符动态生成内容
- 复杂的多视频片段编排

## 快速开始

### 1. 创建简单模板

1. 访问 `/templates/unified`
2. 点击"新建模板"
3. 确保在"简单模式"标签页
4. 填写表单：
   ```
   模板名称：医疗宣教模板
   分类：医疗健康宣教
   片头视频：选择已上传的片头
   片尾视频：选择已上传的片尾
   启用字幕：勾选
   字幕位置：底部
   ```
5. 点击"保存模板"

### 2. 创建高级模板

1. 访问 `/templates/unified`
2. 点击"新建模板"
3. 切换到"高级模式"标签页
4. 填写基本信息和Timeline JSON：
   ```json
   {
     "VideoTracks": [{
       "VideoTrackClips": [
         {"MediaURL": "https://oss.com/header.mp4", "Duration": 3},
         {"MediaURL": "$main_video", "MainTrack": true},
         {"MediaURL": "https://oss.com/footer.mp4", "Duration": 3}
       ]
     }]
   }
   ```
5. 点击"保存高级模板"

### 3. 模式转换

**简单 → 高级**（自动生成Timeline）：
1. 在简单模式填写完表单
2. 点击右侧"切换到高级模式"按钮
3. 系统自动生成Timeline JSON
4. 可以在高级模式下继续编辑

**高级 → 简单**（解析Timeline）：
1. 在高级模式编辑Timeline JSON
2. 点击右侧"切换到简单模式"按钮
3. 系统尝试解析并填充简单字段
4. 注意：复杂的Timeline可能无法完全解析

## 占位符系统

高级模板支持以下占位符：

| 占位符 | 说明 | 示例值 |
|--------|------|--------|
| `$main_video` | 主视频URL | `https://oss.com/video.mp4` |
| `$mainSubtitleDepart` | 医院科室（竖排） | `青岛大学\n附属\n医院` |
| `$mainSubtitleName` | 医生姓名职称（竖排） | `张医生\n主任` |
| `$beginingSubtitleTitle` | 视频标题 | `白血病健康宣教` |
| `$beginingAudioTitle` | TTS语音标题 | `白血病健康宣教` |

**使用示例**：
```json
{
  "VideoTracks": [{
    "VideoTrackClips": [
      {"MediaURL": "$main_video", "MainTrack": true}
    ]
  }],
  "SubtitleTracks": [{
    "SubtitleTrackClips": [{
      "Type": "Text",
      "Text": "$mainSubtitleDepart"
    }]
  }]
}
```

## 快速模板

高级模式提供3个快速模板：

### 1. 标准模板
```
片头 → 主视频 → 片尾
```

### 2. 主视频优先
```
主视频 → 片头 → 片尾
```

### 3. 主视频最后
```
片头 → 片尾 → 主视频
```

点击对应按钮即可插入模板JSON。

## 实时预览

右侧面板提供实时Timeline JSON预览：

- **简单模式**：根据表单自动生成Timeline
- **高级模式**：显示当前编辑的Timeline

## 模板管理

### 查看模板列表
- 点击"模板列表"按钮
- 显示所有模板（简单+高级）
- 用徽章标识模板类型

### 编辑模板
1. 在模板列表中点击"编辑"
2. 系统自动识别模板类型
3. 加载到对应的模式中

### 删除模板
1. 在模板列表中点击"删除"
2. 确认删除操作
3. 删除后无法恢复

## 工具功能

### JSON格式化
点击"格式化"按钮美化Timeline JSON。

### JSON验证
点击"验证"按钮检查Timeline JSON格式是否正确。

### 重置表单
点击"重置"按钮清空当前表单。

## 常见问题

### Q: 简单模式和高级模式有什么区别？
A: 简单模式通过表单配置，适合标准化场景；高级模式通过JSON编辑，支持灵活的自定义配置。

### Q: 模式转换会丢失数据吗？
A: 简单→高级不会丢失数据，系统自动生成Timeline。高级→简单可能丢失部分复杂配置，建议谨慎操作。

### Q: 占位符在哪里替换？
A: 占位符在视频处理时自动替换，不需要手动修改。系统会根据上传的视频和医生信息自动填充。

### Q: 如何测试模板是否正确？
A: 可以使用"预览Timeline"功能查看生成的JSON，或在实际视频上传时测试。

### Q: 旧模板还能用吗？
A: 完全兼容！访问旧URL会自动跳转到新界面，所有模板数据保持不变。

## 最佳实践

1. **新手**：从简单模式开始，熟悉后切换到高级模式

5. **分类管理**：合理使用分类功能，便于后续查找
