### 6. API测试

#### 6.1 模式转换API测试
```bash
# 测试简单→高级转换
curl -X POST http://127.0.0.1:5000/api/video_templates/convert-demo \
  -H "Content-Type: application/json" \
  -d '{
    "mode": "simple-to-advanced",
    "data": {
      "name": "测试",
      "header_video_url": "http://oss.com/header.mp4",
      "footer_video_url": "http://oss.com/footer.mp4"
    }
  }'

# 测试高级→简单转换
curl -X POST http://127.0.0.1:5000/api/video_templates/convert-demo \
  -H "Content-Type: application/json" \
  -d '{
    "mode": "advanced-to-simple",
    "timeline_json": "{\"VideoTracks\": [{\"VideoTrackClips\": [{\"MediaURL\": \"header.mp4\"}, {\"MediaURL\": \"$main_video\"}]}]}"
  }'
```
